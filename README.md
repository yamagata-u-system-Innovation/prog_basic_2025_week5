## 🧮 課題#3：数当てゲーム

仕様：
- 1〜100の乱数を生成して当てるゲーム
- 最大7回まで挑戦可
- 「もっと大きい」「もっと小さい」とヒントを表示
- 正解したら「正解！」、失敗したら正答を表示
